# 公众号 + 小红书内容创作工作流

> 适用场景：当您有一个内容主题，需要同时生成公众号长文和小红书笔记时使用本模板。

---

## 使用说明

1. 复制以下模板到 Claude 对话中
2. 将 `[你的主题]` 替换为实际内容主题
3. 按顺序执行各步骤
4. 最终输出：公众号长文 + 小红书正文 + 标题备选 + 封面图建议

---

## 工作流模板

```
【内容创作任务】

主题：[你的主题]

请按以下步骤执行内容创作：

---

### 步骤1：主题诊断
使用 /dbs-diagnosis 分析这个主题的受众兴趣度和竞争度，输出诊断报告。

### 步骤2：生成小红书标题
使用 /dbs-xhs-title + [你的主题]，生成3个小红书爆款标题备选。

### 步骤3：写公众号长文
使用 /article-writing 写一篇 [你的主题] 的公众号长文（1500-2500字），标题为 [选定的标题]。

### 步骤4：内容诊断
将步骤3的文章输入 /dbs-content 进行诊断，获取改写建议。

### 步骤5：改写（如需要）
根据步骤4的建议，使用 /article-writing 改写文章。

### 步骤6：改写小红书版本
将最终公众号文章输入 /dbs-content，请求改写成小红书版本（800字内，风格活泼）。

### 步骤7：生成小红书标题
使用 /dbs-xhs-title + 小红书正文，生成10+个爆款标题备选。

### 步骤8：生成封面图
使用 /claude-design 为 [公众号标题] 生成封面图设计建议或HTML页面。

---
```

## 快速执行版

如果已完成主题调研，可直接使用简化版：

```
【内容创作】

主题：[你的主题]

步骤：
1. /dbs-xhs-title [你的主题] → 3个标题备选
2. /article-writing [你的主题]，1500-2500字公众号长文
3. /dbs-content [文章] → 改写建议
4. /article-writing 改写版本
5. /dbs-content [公众号文] → 改写成小红书版本（800字内，活泼风格）
6. /dbs-xhs-title [小红书正文] → 10+爆款标题
7. /claude-design [公众号标题] → 封面图

最终输出：公众号长文 + 小红书正文 + 10+标题 + 封面图
```

---

## 技巧提示

- **标题优化**：小红书标题建议包含数字、情绪词、悬念词
- **封面图**：使用 /claude-design 生成 HTML 原型，截图后可用 Canva 套版
- **多平台适配**：公众号偏向深度和专业，小红书偏向轻松和视觉冲击
- **发布时间**：建议周二至周四 19:00-21:00 发布效果较好
