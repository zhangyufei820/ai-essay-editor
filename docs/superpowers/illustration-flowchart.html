安装流程 · 6 步完成
1
管理员打开 PowerShell
搜索 PowerShell → 右键 → 以管理员运行
✓
2
安装 WSL2
wsl --install → 重启电脑
✓
3
开启网络加速工具的局域网权限
Surge / Stash / Clash 设置中开启 Allow LAN
✓
4
WSL 终端设置网络通道
export host_ip=$(grep nameserver ...) && export https_proxy=...
✓
5
装工具 + 下载 Hermes Agent
apt install python3 pip git → git clone → pip install -e .
✓
6
写进 ~/.bashrc，以后输入 hm 启动
配置永久别名和网络环境变量
✓
⚠️ 第 3 步最常被忽略——跳过这步后面会一直超时
