## Introduction

**Essay AI Suite** is a shared AI service suite for Dify apps. It moves essay grading, batch jobs, OCR, and document extraction out of complex Dify workflows into stable HTTP services.

## Features

- **Lightweight Dify Integration**: Provides HTTP endpoints for single essay grading, batch grading, OCR, and document extraction.
- **Stable Batch Jobs**: Includes job status, progress, retries, timeouts, and persisted results.
- **Direct Provider Calls**: Supports OpenAI-compatible LLMs, vision OCR models, and external document extraction APIs.
